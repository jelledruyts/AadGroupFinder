Purpose

This application was built out of the simple frustration that it's often too hard to find discussion lists on a certain topic.

In the spirit of the growth mindset — and because all the necessary components are available in the magic of the cloud — we set out to provide a simple but effective solution.

With this application, you can't just search for groups based on name and description, but also on community-provided annotations (notes and tags). This way, we can all help each other find the most relevant groups.

We also wanted to make it easier to find relevant groups, based on shared membership of certain people or groups.

How It Works

Using the Azure AD Graph API, a WebJob hosted in an Azure Web App processes differential queries to synchronize the mail-enabled groups into an Azure Search index.

An ASP.NET Core Web API that is hosted in the same Azure Web App uses the Azure AD Graph API and the Azure Search index to expose the core functionality to its client applications.

The web application you are using right now is an AngularJS based Single-Page Application written in TypeScript that signs you in to the corporate directory and allows you to work with the Web API to easily and quickly find what you are looking for...

The code is all open source — we welcome any contributions or issue reports!
